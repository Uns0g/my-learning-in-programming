Desenvolvimento Web

O Desenvolvimento Web é o nome dado ao processo de desenvolvimento de um website, uma página, ou uma aplicação web para a rede global (internet), ou para uma rede local.

As ferramentas

O começo deste tópico vai focar em front-end porque é bem mais fácil menos difícil do que ir direto para o lado do servidor1. Os desenvolvedores front-end utilizam o HTML para estruturar páginas, o CSS para estilizar e deixar a página visualmente compreensível para o visitante e alguma outra linguagem de programação para web2 para possibilitar a interação do usuário com a página.

Desenvolvedores mais experientes, após dominarem HTML, CSS, uma linguagem de programação para web (normalmente o JavaScript), utilizam frameworks e/ou bibliotecas para facilitar e agilizar o trabalho de desenvolver uma aplicação web ou um site e também desenvolver o back-end da aplicação ou site, você provavelmente vai encontrar as seguintes ferramentas deste tipo se navegar pela web:

  • Bootstrap - CSS
  • Materialize - CSS
  • Node.js - JS (back)
  • ASP.NET - C# (back)
  • Django - Python (back)
  • React - JS biblioteca (front)
  • Laravel - PHP (back)

Obviamente essas são apenas alguns exemplos, existem vários frameworks e bibliotecas para praticamente qualquer linguagem web que existe.

Como Funciona a Web

Por onde começar?

Depende do que você quer, mas no geral a maior parte dos desenvolvedores começam aprendendo como funciona a "containerização" do HTML, depois partem para o CSS e aprendem o que é mais usual na estilização e partem para alguma linguagem de programação, normalmente o JS (JavaScript) porque tem mais tutorial no YouTube é a mais utilizada para programação front.

Nota 1
O lado do servidor é o dito back-end, diferentemente do front, o back se foca não no cliente, mas no servidor. Enquanto o front cuida da parte visual do site/aplicação o back cuida da organização dos dados armazenados no servidor, segurança, autenticação e autorização de usuários, administração do banco de dados, manipulação de dados e tantos outros termos a mais que eu, como um "front-end developer" não tenho criatividade para nomear.
Nota 2
HTML e CSS não são linguagens de programação, HTML é uma linguagem de marcação (a única deste tipo) de hipertexto (sei lá o que é isso), tanto é que HTML é acrônimo para HyperText Markup Language e CSS é uma linguagem de estilização de elementos de documentos (documentos HTML no caso).
