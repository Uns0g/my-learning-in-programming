HTML & CSS Básico

Esta seção do Aprender é focada em ensinar o mais básico de HTML e CSS, se você já sabe as tags mais comumente usadas e sabe fazer exercício de colorir para criança as estilizações mais básicas com CSS pule esta parte e avançe para o intermediário, onde aprenderá mais sobre o funcionamento de tags específicas e estilizações um pouco mais avançadas, como Grid e FlexBox e Media Queries.

O HTML

O HTML é uma linguagem de marcação de texto, ou seja ela marca o texto de um documento/arquivo de... texto, diferenciando-os por tags, que servem justamente para a marcação.

O HTML é usado para estruturar páginas da web (que são também documentos). Ele que realmente cria quase tudo o que se vê no site e dá nome ao que é criado, sendo a primeira coisa que é carregada pelo seu navegador quando você abre uma página.

O que você vai aprender

Novamente o que você aprenderá como funciona a estruturação de uma página e como você pode estruturar as suas da maneira mais básica possível, além de saber as tags mais comuns na construção de um site.

Também já aprenderá a nomenclatura que talvez seja a mais usada pelos desenvolvedores, o BEM, acrônimo para Block Element Modifier ou em português "Bloco Elemento Modificador" para poupar você de uma das tarefas mais chatas que um desenvolvedor front-end tem que fazer, nomear classes de uma maneira compreensível.

Já aqui você aprenderá como funciona a sintaxe do HTML

Sintaxe E Funcionamento

A sintaxe1 do HTML é muito simples e seu funcionamento ainda mais simples.

A tag (<>) define que aquele texto digitado no documento HTML e o vai ser "marcado", classificado de um certo modo, ou seja, indica que aquilo é um elemento e o conteúdo da tag <> define qual é o tipo do elemento, com o que efetivamente ele irá ser "marcado".

Os elementos no código html são como caixas dentro da tela, tanto é que se você já selecionou algum texto em uma página (se você tem entre 12 e 72 anos de idade certamente já fez isso) percebe que um retângulo é criado ao redor do texto que você seleciona, isso já mostra que até mesmo o texto é uma "caixa" no HTML e o tamanho dela (caixa) é definido pelo espaço que o conteúdo ocupa nela.

Tirando algumas tags específicas que não convém mencionar agora, todas as tags HTML tem um começo e um fim definidos, por exemplo a tag para este texto é a tag <p> que indica que o texto faz parte de um parágrafo, essa tag começa com <p> e termina com </p>.

Desse modo você obtem um elemento dentro de outro, uma "caixa" dentro de outra "caixa" e vai construindo estruturas cada vez mais complexas dentro do seu site.

CSS

O CSS é usado para maquiar dar um estilo para a página. Ele é o responsável por dar cores, definir fontes diferentes, o tamanho delas, modificar a posição dos itens na tela (ou em um container específico), dar bordas e sombras, modificar links, desaparecer e aparecer elementos, fazer animações e também pode definir tamanhos de elementos. Enfim... o CSS muda a maneira como os elementos da página são mostrados na tela.

O CSS é a segunda coisa que o navegador carrega de uma página web, vem logo depois do HTML, por isso as vezes você nota um delay para alguns itens de uma página se chegarem a sua forma final.

Se você seguir a tabela de tópicos em sequência (o que não é necessário, já que eu mesmo aprendi sem uma ordem) verá que só vai começar a ver CSS depois de alguns tópicos de HTML, mas como eu disse não é necessário e se quiser consultar algum tópico de CSS vai pro site da W3Schools é só clicar no tópico desejado.

O Que você vai aprender

Você vai aprender como funciona o CSS logo na próxima seção desta mesma página, mas nessa parte mais básica de CSS você vai aprender a mudar cores dos elementos, mudar o tamanho dos elementos disposição dos elementos na tela ou no container, mudar links, aparecer e desaparecer elementos, aprender a sintaxe do CSS e vai fazer tudo isso enquanto constrói um site para dispositivos desktop Fala computador logo!

Como Funciona O CSS

CSS significa Cascading Style Sheet o que em português pode ser traduzido para Folhas de Estilo em Cascata. O nome já mostra como funcionam os arquivos/documentos de texto CSS. São folhas (arquivos) de estilização de elementos HTML, estilização essa aplicada em cascata.

Procurando o significado de cascata na web, achei uma definição do dicionário Merriam-Webster que cabe muito bem ao que quero expor aqui, a definição é a seguinte:

attachment Algo organizado ou que ocorra numa série ou sucessão de estágios de modo que cada estágio deriva ou age sobre o resultado do anterior.
traduzido do inglês

Esse efeito cascata do CSS está diretamento ligado à estrutura do HMTL, a estilização é aplicada a um elemento por seleção e se o elemento selecionado possui "filhos" (ver herança) ALGUMAS PROPRIEDADES serão aplicadas, essas propriedades são o tipo de estilização que será aplicada, se você quer alterar o tamanho do texto de um elemento por exemplo terá que usar a propriedade font-size2 e os elementos "filhos" a esse estilizado, por não terem o seu próprio tamanho de fonte definido ainda, herdarão do "pai" o tamanho.

Sintaxe do css

A sintaxe do CSS é também de fácil entendimento.

Como disse, o CSS funciona por seleção e herança. O elemento selecionado é chamado de seletor3 e esse seletor recebe atributos ou propriedades como queira chamar, que, como também disse, são os tipos de estilização e os atributos recebem valores que são... valores.

Para ficar mais fácil de entender veja o exemplo logo abaixo.

seletor{
atributo: valor; }

A sintaxe toda do CSS é basicamente essa, pouca coisa sofre algumas alterações, por exemplo os pseudo-elementos e pseudo-classes (que são tópicos mais avançados) são declarados logo após os seletores, sendo que os pseudo-elementos são identificados pelo sinal de dois pontos repetido :: e as pseudo-classes identificadas pelo sinal de dois pontos :

seletor::pseudo-elemento{
atributo: valor; }
seletor:pseudo-classe{
atributo: valor; }

Há também as variáveis, funções e animações (isso mesmo, funções e variáveis no CSS) que tem uma maneira de ser escrita um pouco diferente, mas não alteram nada na estrutra original do CSS.

Nota 1
Se você não sabe ABSOLUTAMENTE NADA de programação, a sintaxe é a maneira como uma linguagem é escrita, da mesma maneira que uma língua escrita tem a sua sintaxe as linguagens também têm as suas próprias, que normalmente são bem menos difíceis e a maioria das linguagens seguem padrões para suas estruturas, no caso, como HTML e CSS são linguagens singulares e têm sintaxes também singulares.
Nota 2
A propriedade font-size é utilizada para alterar o tamanho da fonte no conteúdo do elemento, ela pode receber valores em de tamanho(px e %), valores globais inherit, initial, revert, unset, valores próprios da propriedade font-size e valores em porcentagem (%)
Nota 3
Até agora utilizei o termo elemento para se referir a seletores do css, mas saiba desde já que não usarei mais esse termo de mais fácil entendimento e usarei o termo técnico seletor, porque elemento é o nome dado a qualquer coisa dentro da página, já seletor é algo específico do CSS, é o que seleciona um elemento da página
