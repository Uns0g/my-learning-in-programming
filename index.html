Soft Skills

(Criatividade, Proatividade)

A coisa mais divertida da programação é o despertar de um espírito criativo que é escondido de todos nós desde a infância, uma predisposição que todos têm para conectar conceitos, identificar padrões e procurar criar alguma solução para o problema que enfrenta, é impossível fazer isso sem "ir lá e encarar" bater de frente com o problema e enfrentá-lo.

O que é mais procurado hoje em programação é a tal da "inovação" e a vontade de oferecer não só soluções para o problema, mas a melhor solução para o problema. É exatamente isso que eu ofereço.
